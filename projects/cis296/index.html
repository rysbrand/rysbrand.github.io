CIS-296 — Azure File Manager

View on GitHub →
C# · ASP.NET Core · Azure · Razor Pages

A cloud-connected file management web application built for CIS-296 Advanced MS Web Development. The app uses ASP.NET Core Razor Pages with ASP.NET Identity for role-based authentication, and integrates with Azure Blob Storage for file uploads and retrieval.

Features

  • Azure Blob Storage Integration — Files are uploaded to and retrieved from Azure Blob Storage via a dedicated service layer.
  • Role-Based Authorization — Admin users can upload files; all authenticated users can view the file list. Roles are seeded automatically on startup.
  • ASP.NET Identity — Full user authentication system with login, registration, and role management built on top of Entity Framework Core.
  • Razor Pages Architecture — Clean separation of UI and page model logic using ASP.NET Core's Razor Pages pattern.
  • SQL Server Backend — User and identity data stored in Azure-hosted SQL Server via Entity Framework Core migrations.

Sample Code

The BlobService handles all Azure Blob Storage operations:

public async Task UploadFileAsync(IFormFile file)
{
    var blobClient = _containerClient.GetBlobClient(file.FileName);

    using var stream = file.OpenReadStream();
    await blobClient.UploadAsync(stream, overwrite: true);
}

public async Task<List<string>> GetFileNamesAsync()
{
    var files = new List<string>();

    await foreach (var blobItem in _containerClient.GetBlobsAsync())
    {
        files.Add(blobItem.Name);
    }

    return files;
}

The upload page is restricted to Admin role only:

[Authorize(Roles = "Admin")]
public class UploadModel : PageModel
{
    private readonly BlobService _blobService;

    [BindProperty]
    public IFormFile? UploadFile { get; set; }

    public async Task<IActionResult> OnPostAsync()
    {
        if (UploadFile == null || UploadFile.Length == 0)
        {
            ModelState.AddModelError(string.Empty, "Please choose a file.");
            return Page();
        }

        await _blobService.UploadFileAsync(UploadFile);
        return RedirectToPage("/Files");
    }
}

Roles and an admin user are seeded automatically on startup:

string[] roles = { "Admin", "User" };

foreach (var role in roles)
{
    if (!await roleManager.RoleExistsAsync(role))
    {
        await roleManager.CreateAsync(new IdentityRole(role));
    }
}
