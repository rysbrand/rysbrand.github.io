CIS-241 Dynamic Website Development

View Live Site →
PHP · MySQL · MVC · PDO

A dynamic, database-driven web application built solo for CIS-241. The app implements user registration and authenticated login using PHP sessions, PDO database access, and the MVC pattern.

Features

  • User Registration — New users can register with email, password, and name. Passwords are hashed using PHP's built-in password_hash() before being stored.
  • Authenticated Login — Users log in with their email and password. Credentials are verified against the database using password_verify().
  • Session Management — PHP sessions maintain login state across pages, protecting dashboard and account routes from unauthenticated access.
  • Account Management — Logged-in users can view and update their account information.
  • PDO Database Access — All database queries use PHP Data Objects (PDO) with prepared statements to prevent SQL injection.
  • MVC Structure — Code is organized into models, views, and a controller (index.php) for clean separation of concerns.

Tech Stack

  • PHP — Server-side logic and routing
  • MySQL — Relational database for user accounts
  • PDO — Secure database access with prepared statements
  • HTML/CSS — Front-end structure and styling
  • InfinityFree — Live PHP/MySQL hosting

Live Demo

The application is hosted live at dwdunit3.infinityfreeapp.com. You can register a new account or log in to explore the dashboard and account management features.
